Services

Customized bots that will be ready to be deployed within a month. Currently, we are focusing on 3 bots over a period of 3 months viz. Car Insurance, Interactive food / beverage ordering, and Travel Assistance. Soon, we will also focus on building our own digital assistant – especially targeted for the Indian urban markets.

chatbot design

Our model consists of the Interface, the Recommendation Engine and the Conversation Engine along with a smart database.

recommendation engine

The recommendation engine will use the processed input data to generate appropriate responses taking into account various factors like intent, context etc. This is where the learning takes place. If the recommendation engine comes across new data, the learning algorithms will dynamically modify the database to account for the new transaction.

billing

The billing engine (integrated into AWS) will keep count of queries/hits on each bot. A billing model will be developed to charge taking into account these queries.

customizability

Each individual bot can access the learning and conversation engines individually (with a specific ID).

baba da dhaba

Currently, we're working on a smart, interactive chat-bot for one of the top restaurants in Bangalore - Baba da Dhaba.

database

The menu of the respective restaurant is fed to the engine. A smart database is created for that restaurant.

customized for each customer

Given the feature set of the customer, each food item in the menu is give a probability score.

recommendations

This enables the bot to make the best recommendation possible with the available data and the current context.
